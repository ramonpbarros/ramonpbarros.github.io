Daily Scheduler

Simple calendar application that allows the user to save events for each hour of the day. This app will run in the browser and feature dynamically updated HTML and CSS powered by jQuery. Daily planner sets (on the list) a different color background according to time of the day. Activities set on the planner will be saved on local storage. Displays the current day, month, and date on the top of the page.
